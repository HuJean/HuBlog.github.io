# Jekyll+Github构建个人博客

## Jekyll基本用法
  * 安装
    * Mac

    ~~~
    ~ $ gem install jekyll
    ~ $ jekyll new myblog
    ~ $ cd myblog
    ~/myblog $ jekyll serve

    此时，访问http://localhost:4000即可。
    ~~~

    * windows (https://blog.csdn.net/mouday/article/details/79300135)
      * 安装 Ruby 
      * 安装 DevKit 
      * 安装 Jekyll
